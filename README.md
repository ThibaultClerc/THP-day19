## TIC TAC TOE RUBY 

Here is a version of the famous game tic tac toe.
We made it by using the OO programming.

It contains multiple files each containing a class embodying an object :
  - application
  - game
  - board
  - player
  - board_case

## To run the game:
  - launch 'bundle install'
  - launch 'ruby app.rb'
